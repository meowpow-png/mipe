# Examples

## Purpose

Examples demonstrate how Codex Devkit is intended to be used in real projects.

Each example serves three purposes:

- Validate that Devkit can be used to develop a project from start to finish
- Provide a complete reference implementation
- Teach the recommended development workflow through a curated transcript

## Available Examples

### Password Generator CLI

A command-line application for generating secure passwords and passphrases.

Focus areas:

- CLI development
- Testing
- Configuration
- Input validation

### Weather Client

A command-line client for retrieving and displaying weather forecasts from a public API.

Focus areas:

- HTTP APIs
- JSON processing
- Error handling
- External integrations

## Creating New Examples

New examples should be:

- Small enough to understand in a single session
- Focused on a single domain or problem
- Interesting enough to exercise realistic development workflows
- Accompanied by a curated transcript demonstrating how the project was developed
